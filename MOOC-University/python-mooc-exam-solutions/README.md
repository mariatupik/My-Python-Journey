# University of Helsinki: Python Programming MOOC Exam Solutions

This folder contains my solutions for the Python Programming MOOC examination. The exercises focus on algorithm design, data manipulation, and modular programming.

## 🧠 Challenges Covered:

### 1. Character Frequency Analyzer (`Exercise1.py`)
- **Objective**: Analyze user-input strings and count character occurrences.
- **Logic**: Implemented a frequency dictionary, filtered results based on a minimum threshold, and sorted data in descending order of occurrence.
- **Key Concepts**: `while` loops, `dict` manipulation, list sorting, and formatted string output.

### 2. Matrix Deep Copy & Transformation (`Exercise2.py`)
- **Objective**: Manage complex nested lists (matrices) without mutating the original data.
- **Logic**: Developed custom `copy` and `flip` functions. Demonstrated a deep understanding of the difference between object equality (`==`) and object identity (`is`) in Python.
- **Key Concepts**: Deep vs. Shallow copying, matrix traversal, and pointer/reference management.

### 3. Fraction Arithmetic Calculator (`Exercise3.py`)
- **Objective**: Create a robust calculator for fractional numbers.
- **Logic**: Utilized the built-in `fractions` module. Implemented string parsing to separate numerators, denominators, and operators.
- **Key Concepts**: String splitting (`split()`), type conversion, and handling mathematical operations as modular functions.
