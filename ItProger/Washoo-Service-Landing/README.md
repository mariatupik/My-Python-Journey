## 🛠 Key Features & Functionality
* **Fixed Interactive Header:** A smart navigation bar that dynamically switches to `position: fixed` based on scroll position to provide constant access to site sections.
* **Slide-out Sidebar Menu:** A custom-engineered mobile-friendly navigation menu powered by **jQuery** animations.
* **Parallax Architecture:** Advanced use of `background-attachment: fixed` and `background-blend-mode` for a dynamic visual depth.
* **Dynamic Carousel:** A responsive, touch-enabled slider implemented via **Swiper.js** for showcasing core services and technologies.
* **Smooth Scroll Navigation:** Integrated "Back to Top" functionality and smooth document transitions for enhanced UX.
* **Modern Typography & Icons:** Leveraging **Google Fonts (Montserrat)** and **Font Awesome 6** for a clean, professional aesthetic.

## 💻 Tech Stack
* **Core:** HTML5, CSS3 (Flexbox, Fixed positioning, Parallax effects).
* **Scripts:** JavaScript (ES6+), **jQuery** (DOM manipulation & animations).
* **Libraries:** * [Swiper.js](https://swiperjs.com/) (Slider Engine)
    * [Font Awesome 6](https://fontawesome.com/) (Vector Icons)
    * [Google Fonts](https://fonts.google.com/)

## 📐 Development Insights
This project served as a practical application of **event-driven programming** in the browser. Key challenges included managing the Z-index hierarchy for the hidden menu and ensuring the parallax effects remain performant across different viewport sizes.
